Questão 2

Qual é a diferença entre SQL e NoSQL? Dê exemplos de cada um.

a) SQL é uma linguagem de consulta estruturada usada principalmente em bancos de dados relacionais, enquanto NoSQL é uma linguagem de programação utilizada para criar aplicativos web.
b) SQL é uma linguagem de programação usada para manipular dados em bancos de dados relacionais, enquanto NoSQL é um tipo de banco de dados que não segue um modelo relacional.
c) SQL e NoSQL são termos intercambiáveis e referem-se ao mesmo tipo de banco de dados.
d) SQL é uma tecnologia de banco de dados usada em aplicativos de desktop, enquanto NoSQL é usada em aplicativos móveis.
e) SQL é um banco de dados relacional, enquanto NoSQL é um banco de dados baseado em gráficos.

Questão 3

O que é normalização em banco de dados e por que é importante?

a) Normalização é o processo de aumentar a redundância nos dados, a fim de melhorar o desempenho do banco de dados.
b) Normalização é o processo de otimização do banco de dados para melhorar a velocidade das consultas.
c) Normalização é o processo de organizar os dados em uma estrutura lógica para evitar redundâncias e inconsistências, garantindo a integridade dos dados.
d) Normalização é o processo de adicionar mais dados ao banco de dados para torná-lo mais completo.
e) Normalização é o processo de diminuir a complexidade do banco de dados, removendo tabelas desnecessárias.
